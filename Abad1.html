موقعي التجريبي

معسكر البرمجة الشامل

خطة التدريب

  1. SQL (مدة: أسبوع)
    مقدمة عن قواعد البيانات وأنظمة إدارة قواعد البيانات
    أنواع قواعد البيانات: العلائقية (Relational) مقابل غير العلائقية (NoSQL)
    إنشاء وتعديل الجداول (CREATE, ALTER, DROP)
    الاستعلامات الأساسية واسترجاع البيانات (SELECT, WHERE, ORDER BY, GROUP BY, HAVING)
  2. لغة C# (مدة: أسبوع)
    التعرف على بنية البرنامج
    كتابة أول برنامج بلغة C#
    أنواع البيانات والمتغيرات والثوابت
    الجمل الشرطية والحلقات
  3. البرمجة الكائنية (OOP) باستخدام C# (مدة: أسبوع)
    الفئات والكائنات
    الوراثة، التعددية الشكلية، التغليف والتجريد
    الواجهات والفئات المجردة
  4. HTML/CSS/JavaScript + Bootstrap (مدة: 3 أسابيع)
    HTML5: بناء الصفحات والعناصر الأساسية
    CSS3: النماذج الصندوقية والتصميم المتجاوب
    JavaScript: التفاعل مع DOM ومعالجة الأحداث
    Bootstrap: المكونات الجاهزة والتصميم الشبكي
  5. Entity Framework Core (مدة: أسبوع)
    مقدمة في ORM
    تنفيذ العمليات CRUD باستخدام EF Core
    استخدام LINQ للاستعلام
  6. ASP.NET Core MVC (مدة: أسبوعان)
    مقدمة في بنية MVC
    التعامل مع النماذج والتحقق من صحة المدخلات
    المصادقة وإدارة الصلاحيات
  7. ASP.NET CORE WEB API (مدة: أسبوعان)
    مقدمة في Web API
    التحكم في البيانات والتوجيه (Controllers & Routing)
    التعامل مع البيانات باستخدام Entity Framework Core
    التحقق من صحة البيانات وإدارة الطلبات
    توثيق الـ API والاختبار
  8. الهيكلية النظيفة (Clean Architecture) (مدة: أسبوع)
    مبادئ الهيكلية النظيفة وفصل المسؤوليات
    استخدام أنماط الخدمة والمستودع
  9. Flutter (مدة: 4 أسابيع)
    بناء التطبيقات باستخدام Flutter
    إدارة الحالة باستخدام Provider وGETX
    تكامل API والعمل مع Firebase
